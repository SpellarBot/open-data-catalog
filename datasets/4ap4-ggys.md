## Description

Building roof outlines derived from 2009 orthophotography. The dataset can be dowloaded by clicking the link on the upper right corner in this page https://data.seattle.gov/dataset/2009-Building-Outlines/y7u8-vad7   
Note - these data do not contain peak elevation data. For peak elevation data you can refer to the 1999 building outlines via this link:  https://data.seattle.gov/Land-Base/CGDB-BLDG1999-PLGN/x87c-yyrn
