## Description

The ?For The Record? publication shows the status of the projects contained in the annual program of road and structure projects from the past fiscal year. There are five layer files that make up the program. They are the roadway, spot, structure, engineering, and land acquisition files. This publication will list the same Improvement Types as the Annual Publication. The Spot Layer File is populated by selecting project records containing a Spot Improvement Type.
