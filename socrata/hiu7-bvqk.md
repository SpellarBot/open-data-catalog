## Description

The Special Use Districts are a component of the Zoning Map which in turn is a key component of the San Francisco Planning Code. The Zoning Map comprises: - Zoning Districts - Height and Bulk Districts - Special Use Districts - Preservation Districts - Coastal Zone Area - Special Sign Districts The official Zoning Map can be found in the San Francisco Planning Code: http://planningcode.sfplanning.org (click on the links under ZONING MAPS on the left navigation column).
